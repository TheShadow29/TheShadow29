## Philosophy

> Good research should make the problem clearer, not just the benchmark higher.

I care about work that grounds language in the world it describes rather than treating vision and language as loosely correlated signals. A paper should sharpen the question, a benchmark should measure something real, and released code should help other people reproduce, stress-test, and extend the result. The goal is not only to publish models, but to leave behind research artifacts that are precise, useful, and durable.
